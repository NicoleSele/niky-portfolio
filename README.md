- **Diseño Responsivo**: Optimizado para dispositivos móviles, tablets y escritorio

## 📝 Personalización

### Cambiar Colores

Los colores principales están definidos en las variables CSS al inicio del archivo `<style>`:

```css
:root {
    --primary: #2c3e50;      /* Azul oscuro */
    --secondary: #e74c3c;    /* Rojo */
    --accent: #f39c12;       /* Naranja */
    --light: #ecf0f1;        /* Gris claro */
    --dark: #1a1a1a;         /* Negro */
}
```

### Agregar Proyectos

En la sección de portafolio, copia y modifica un `portfolio-item`:

```html
<div class="portfolio-item">
    <div class="portfolio-image">Tu Proyecto</div>
    <div class="portfolio-content">
        <h3>Nombre del Proyecto</h3>
        <p>Descripción del proyecto</p>
        <span class="portfolio-tag">Etiqueta</span>
    </div>
</div>
```

## 📱 Responsive Design

El portafolio se adapta automáticamente a diferentes tamaños de pantalla:
- **Desktop**: Diseño completo con múltiples columnas
- **Tablet**: Ajuste de espacios y tipografía
- **Mobile**: Diseño de una columna optimizado
